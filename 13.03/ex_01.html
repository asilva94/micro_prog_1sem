Exercicio 1

Crie uma página onde o usuário irá digitar dois números. Então use uma rotina de repetição para exibir
todos os números existentes entre os dois números, de forma negativa, ou seja, contando do maior para o menor.
Utilize estruturas condicionais conforme necessário para conseguir exibir corretamente do maior para o menor,
independentemente de qual número o usuário digitou primeiro.
